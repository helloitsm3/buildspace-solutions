### Windows

```sh
1. cd frontend
2. npm install
3. type nul >> ".env"       # This will create a .env file
```

### Linux or MacOS

```sh
1. cd frontend
2. npm install
3. touch .env               # This will create a .env file
```

```sh
# This should be in your .ENV file
OPENAI_API_KEY=YOUR_OPENAI_API_KEY
```
